Javascript in Web Development

JavaScript is a popular programming language for many reasons. It is not only easy to learn, it is a client-side language that can be executed quickly. Unfortunately, while it may have its strengths, there are some weaknesses, such as security that prevent it from being wholly reliable.

Here is a breakdown of some of the strengths and weaknesses of JavaScript.

Strengths Description
JS can change HTML and CSS By changing the Document Object Model, JS can change the HTML and CSS of existing web pages, making web pages interactive
JS code is executed quickly JS is a client side language, meaning that it is written into the HTML and executed by the browser. Since it is not accessed through the web server, it generally loads quickly.
JS code is easy to learn JS is an object-oriented language and has many similarities to languages like Java and Ruby, making it easy for programmers to learn


Weaknesses Description
JS can be accessed by anyone Since JS is written into an HTML page, it can be accessed by anyone who is viewing the page
JS can be unreliable Occassionally, JS can be rendered differently on different browsers, making it unreliable and inconsistent
